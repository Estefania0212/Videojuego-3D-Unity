# 🌌 Celestial Hops - 3D

Videojuego de aventura y exploración espacial desarrollado en Unity 3D.

## 🚀 Descripción
Versión 3D del juego Celestial Hops, donde el jugador explora entornos inmersivos mientras enfrenta desafíos y descubre nuevos mundos.

## 🎮 Características
- Exploración en 3D
- Entornos interactivos
- Mecánicas de movimiento
- Escenarios espaciales

## 🛠️ Tecnologías
- Unity 3D
